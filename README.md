<h1>API Node.js </h1>

## Requeriments

* [Nodejs](https://nodejs.org/) - Javascript
* [MySQL](https://www.mysql.com) - Data Base

## Install dependencies

```bash
$ npm install
```

⚠️ Must have local mysql database installed. Check MySQL. Check database credentials.

## Run API

```bash
$ npm run dev
```

## API will be listening at

```bash
$ http://localhost:3000
```
